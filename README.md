# Bhajan

Три родственных проекта в одном репозитории:

| Папка | Что это | Кому |
|---|---|---|
| корень (Next.js) | **Bhajan Sangam** — веб для сообщества: общая база редактируемых аккордовых листов, публикация, шаринг. | Все ищущие / участники |
| [`audio-service/`](audio-service/) | Python + FastAPI: mp3/YouTube → стемы, MIDI, аккорды. Общее ML-ядро, используется веб-приложением и десктопом. | Внутренняя служба |
| [`desktop/`](desktop/) | **Bhajan Desktop** — локальное Mac-приложение: стемы, MIDI, аудиоредактор с секциями, Synthesia-плеер, обучалка без повторов. | Разучивающие / певцы |

Видения по каждому — в `PRODUCT.md` соответствующих папок.

Ветка активной разработки: `claude/funny-heisenberg-p0nvup`.
