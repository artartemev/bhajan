# Bhajan Desktop — видение

## Что это

Локальное Mac-приложение для **обучения** играть/петь бхаджаны и киртаны по
записи. Всё крутится на твоей машине: тяжёлые модели (MuScriptor large и т.п.)
живут на диске, ничего не уходит наружу.

Не путать с [Bhajan Sangam (веб)](../README.md): тот — общая база
редактируемых **аккордовых листов** для сообщества. Desktop — личный
инструмент разучивания. Пересекается только в одну сторону: Synthesia-плеер,
который здесь появится первым, потом добавится в веб как отдельная фича.

## Что делает

1. **Импорт трека**: mp3, YouTube (ссылка → yt-dlp), из библиотеки Bhajan Sangam.
2. **Разложение**: стемы (vocals / instrumental / harmonium) через Demucs,
   MIDI на инструмент через MuScriptor (веса локально), аккорды.
3. **Аудиоредактор с секциями**: вейвформа, авто-детекция куплет/припев/бридж/
   аутро, ручная правка границ, лейблы.
4. **Synthesia-плеер**: падающие ноты по виртуальной клавиатуре из нашего MIDI,
   синхрон с аудио, темп/транспозиция/цикл-по-секции.
5. **Обучалка «без повторов»**: маршрут проигрывает только **уникальные
   секции**, а не всю песню с повторами. Режимы: медленно / по фразам /
   в темпе / только левая-правая рука.

## Каркас

- **Tauri (Rust shell)** — родное Mac-окно, ~10 МБ shell, автоматические
  обновления, доступ к файловой системе.
- **Frontend (HTML/Canvas)** — Synthesia-плеер на `<canvas>`,
  аудиоредактор на WaveSurfer.js. Возможно позже — React, если разрастётся.
- **Python-сервис внутри** — тот же FastAPI из `../audio-service`, спавнится
  как sidecar (локальный процесс на 127.0.0.1). Переиспользуем всё, что уже
  умеем: Demucs, MuScriptor, аккорды, MIDI.
- **Модели** — кеш в `~/Library/Application Support/Bhajan/models/`,
  менеджер моделей в приложении (скачать/удалить, показать место).

## Этапы

- [~] **Этап 0. Скелет.** Tauri window показывает нашу текущую веб-морду
  из FastAPI-sidecar. Прогонка mp3 работает через существующий пайплайн.
- [ ] **Этап 1. Секционный анализ.** Детекция куплет/припев через
  `librosa.segment.recurrence_matrix` или `msaf`, API отдаёт
  `[{label, start, end}]`.
- [ ] **Этап 2. Аудиоредактор секций.** WaveSurfer.js + regions plugin,
  правка границ, лейблы куплет/припев/бридж/аутро, сохранение.
- [ ] **Этап 3. Synthesia-плеер.** Canvas с падающими нотами из MIDI,
  подсветка активных, темп/транспозиция/цикл по секции.
- [ ] **Этап 4. Обучалка без повторов.** Маршрут «уникальные секции» +
  режимы медленно/по фразам/только-одна-рука.
- [ ] **Этап 5. Менеджер моделей.** UI скачивания MuScriptor small/medium/
  large, статус загрузки, HF-логин из UI, показ занятого места.
- [ ] **Этап 6. Упаковка.** `.dmg`, подпись, автообновления через Tauri
  updater.

## Что осознанно не в скоуп

- **Публикация листов, общая база, шаринг** — это Bhajan Sangam-веб, не
  здесь. Desktop может импортировать/экспортировать через API веба, но
  не хостит.
- **Кроссплатформенность прямо сейчас.** Tauri сам по себе кроссплатформенный,
  но сначала полируем Mac; Windows/Linux — позже, если понадобится.
- **Мобильные приложения.** Отдельная история, не про этот проект.
