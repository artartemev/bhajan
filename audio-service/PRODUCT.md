# Видение продукта

## Что это

Сервис, который из загруженного трека (киртан/бхаджан, в перспективе — любая песня)
делает **понятный аккордовый лист**: текст со стоящими над ним аккордами, разбитый
по секциям. Лист можно **отредактировать** и **опубликовать** — он становится
доступен всем (общая база, как в основном приложении Bhajan Sangam).

ИИ даёт **черновик**, человек его быстро доводит. Ни один автомат не выдаёт
идеальный результат сам — поэтому ручная правка является частью продукта, а не
костылём.

## Приоритеты

1. **Аккордовый лист (ядро).** Чистая последовательность аккордов над текстом, по
   секциям. Тональность песни. Понятно музыканту, а не «178 смен за 8 минут».
2. **Редактор + публикация.** Правка аккорда/позиции/секции, сохранение, шаринг в
   общую базу.
3. **Схемы аккордов** для клавиш / гитары / укулеле (как в Bhajan Sangam).
4. **MIDI «по-настоящему».** Многие играют бхаджаны по нотам — нужна аккуратная
   мелодия вокала и партия фисгармони, в перспективе экспорт в ноты (MusicXML/PDF).
5. **Караоке — сайд-проект.** Для широкой аудитории: «закинул трек → минусовка +
   текст + аккорды». Полезно, но не ядро; строится на тех же артефактах.

## Принципы

- **Не множить неточности.** Раньше складывали 4 неточных ML-шага подряд — на
  выходе шум. Лучше меньше шагов, но надёжных, плюс правка человеком.
- **Музыкальная адекватность важнее «посекундной точности».** Аккорд на такт/долю,
  привязка к тональности — так, как пишут реальные аккордовые листы.
- **Артефакты переиспользуемы.** Один и тот же результат кормит и лист, и схемы
  аккордов, и (сайд) караоке.

## План по этапам

- [~] **Этап 1. Качество аккордов.** Beat-синхронная детекция, оценка тональности,
  1 аккорд на долю/такт, диатоническое смещение (убирает дрожание maj↔min).
  → короткий честный лист. *(в работе)*
- [ ] **Этап 2. Модель аккордового листа + чистый просмотр.** Секции → строки →
  позиции аккордов; тональность; читаемый вид (не караоке).
- [ ] **Этап 3. Редактор + публикация.** Правка/сохранение, публичная база.
- [ ] **Этап 4. Схемы аккордов** для клавиш/гитары/укулеле.
- [ ] **Этап 5. MIDI до ума.** Квантизация по долям, чистая мелодия вокала,
  партия фисгармони, экспорт в ноты (MusicXML/PDF).
- [ ] **Сайд. Караоке-режим** для широкой аудитории (минусовка + текст + аккорды).

## Что намеренно НЕ ядро

Караоке-подсветка по словам — приятный бонус, но не определяет продукт. Точная
посекундная синхронизация текста не нужна для аккордового листа.
